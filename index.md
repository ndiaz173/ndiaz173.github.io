## üì• Download & Install

To download and run the application, follow the steps below.

### Step 1: Visit the Releases Page

Go to the [Releases page](https://github.com/ndiaz173/python-intro-students/releases) to get the latest version. Here, you will find the files necessary for the notebook.

### Step 2: Choose the Notebook File

Locate the **`Introduccion_a_Python_y_Notebooks.ipynb`** file. Click on it to begin the download. This file contains everything you need to start learning Python.

### Step 3: Open the Notebook

1. **Install Jupyter Notebook**: If you haven't already installed Jupyter Notebook, follow these instructions based on your operating system:
   - **Windows**: Install Anaconda, which includes Jupyter Notebook.
   - **macOS**: Use Homebrew to install Jupyter with the terminal command: `brew install jupyter`.
   - **Linux**: You can install Jupyter using pip: `pip install jupyter`.

2. **Run Jupyter Notebook**: 
   - Open your terminal or command prompt.
   - Navigate to the folder where you downloaded the notebook file.
   - Type `jupyter notebook` and press Enter. This will open Jupyter in your web browser.

3. **Select the Notebook**: In the Jupyter interface, click on **`Introduccion_a_Python_y_Notebooks.ipynb`** to open it.

### Step 4: Work with the Notebook

- Follow the instructions in each cell. Enter your code in the provided sections.
- You can run each cell by selecting it and clicking the "Run" button or pressing **Shift + Enter**.

### Step 5: Saving Your Work

After completing your work, save it to ensure you don‚Äôt lose any progress. Use the **File** menu in Jupyter and select **Save and Checkpoint**.

### Step 6: Version Control with Git

If you are familiar with Git, here‚Äôs how to commit your changes:
